## ðŸ§® Methodology

The following analytical methods are applied in this project:

* computation of **daily logarithmic returns**
* estimation of **mean returns and volatility**
* construction of the **covariance matrix**
* calculation of **expected portfolio return and risk**
* exploring **Markowitz's investment portfolio**
* analysation of **PCA-Market index model**
* testing **hypotheses about asset distribution**
* calculation of **Efficient frontier** 
* formation of a **Pareto-optimal portfolio**
* comparison of **expected vs. realized returns**
* visualization of **daily portfolio return dynamics**
